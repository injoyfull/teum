# 틈 (teum) — 숨과 쉼 사이의 편지

감각이 무뎌진 3040을 위한, 2주에 한 번의 안부. 매 호 하나의 감각과 5분이면 충분한 아주 작은 처방 하나.

## 구조

- `index.html` — 구독 랜딩(첫 화면). 굵은 한 줄의 여정 라인 위에 '틈.'
- `letter-01.html` — 1화 웹 매거진 「오늘, 초록을 본 적이 있나요?」(시각 편)
- `email-01.html` — 1화 이메일 버전(보조)
- `assets/` — 이미지 자원

## 배포

GitHub Pages. `index.html`이 첫 화면.

## 상태 (MVP)

- 구독 폼은 아직 발송 플랫폼에 미연결(완료 메시지만 표시)
- 표지 사진(`assets/cover-01-photo.jpg`)을 넣으면 표지에 반영됨(없으면 블루 필드로 표시)
